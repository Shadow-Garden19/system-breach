# SYSTEM BREACH — L'Infection

Jeu de type "crash" en 5×5 : extrais un maximum de multiplicateurs avant de tomber sur le virus.

## Comment jouer

1. **Mise** : Choisis le montant à risquer (boutons −, +, x2, //2).
2. **NB d'explosions** : Choisis le nombre de virus (1 à 24). Plus il y a de virus, plus les multiplicateurs sur les cartes saines sont élevés.
3. **LANCER** : Démarre la partie. Les 25 cartes sont face cachée.
4. Clique sur une carte :
   - **Carte saine** → affiche un multiplicateur (ex. x1.32). Les multiplicateurs s’accumulent.
   - **Carte virus** → partie perdue, mise perdue.
5. **CASH OUT** : Encaisser tes gains actuels (actif après au moins une carte saine).

## Lancer le jeu

Ouvre `index.html` dans ton navigateur (double-clic ou glisser-déposer dans une fenêtre Chrome/Edge/Firefox).

## Fichiers

- `index.html` — structure de la page
- `styles.css` — thème cyberpunk / ville futuriste
- `script.js` — logique du jeu (grille, virus, multiplicateurs, cash out, animation infection)
